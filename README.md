## Installation

#### CocoaPods

```
   platform :ios, '7.0'
   pod "MXMarkdownKeyboard", "~> 0.0.4"
```
#### Without CocoaPods
   just like the Demo 

## Use

1. remeber import 'MXKeyboardToolbar.h'
2. like this
```
   //just a line
    _textView.inputAccessoryView = [MXKeyboardToolbar toolbarViewWithTextView:_textView];
```
